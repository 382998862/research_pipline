可变剪接结果示例

    每个测序样品的Reads与参考基因组比对，通过比对情况确定剪接位置，比较剪接位置的变化从而确定不同的可变剪接类型，具体采用tophat[1]软件进行序列比对及剪接分析，通过splicegrapher[2]软件统计可变剪接类型及作图分析。
