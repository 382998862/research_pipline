网页版报告及分析结果

该目录包括了项目分析结果文件，以及结果的网页展示，具体的目录结构如下：

1.rawdata：项目数据的质量评估
2.geneExpression：样品测序Reads与基因组比对结果数据及样品的表达量数据
3.SNP：SNP分析结果
4.Alt_splice：可变剪接事件分析结果
5.Gene_Structure_Optimize：基因结构优化分析结果
6.NewGene：新基因分析结果
7.DEG_Analysis：差异分析结果
8.index.html：分析结果网页展示主页面
9.html：分析结果网页展示分页面等文件

各分析结果目录的文件说明请见目录内。
